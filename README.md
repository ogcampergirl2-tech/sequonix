# Sequonix

Marketing site for **Sequonix**, an AI-native technology partner building software, AI agents, and automation for organizations across every industry.

## Design

A restrained engineering-studio aesthetic, built to read as senior work rather than a template:

- Flat warm near-black canvas, one disciplined orange accent (no gradients or glows)
- Hairline grid system; sections divided by 1px rules
- Schibsted Grotesk (display + text) with JetBrains Mono for technical labels and section indices
- Quiet motion: short reveal-on-scroll and count-up KPIs only (no scroll-jacking, no animation libraries)
- Sections: hero, capability strip, about, services, industries, work carousel, partnership models, newsletter, contact

Vanilla HTML, CSS, and JavaScript. No build step, no external runtime dependencies.

## Run locally

Open `index.html` in a browser.

## Files

- `index.html`: markup
- `styles.css`: design system, layout
- `script.js`: reveals, count-up, nav, carousel, newsletter validation
