# Sequonix

Marketing site for **Sequonix**, an AI-native technology partner building software, AI agents, and automation for organizations across every industry.

- Futuristic dark, orange-on-black theme
- Space Grotesk + Inter typography
- Kinetic experience: momentum smooth scroll (Lenis), headline mask reveals, animated gradient-mesh background
- Sections: hero, about, services, industries, work (carousel), partnership models, AI newsletter, contact
- Vanilla HTML, CSS, and JavaScript. No build step.

## Run locally

Open `index.html` in a browser.

## Files

- `index.html` — markup
- `styles.css` — theme, layout, animations
- `script.js` — reveals, nav, carousel, newsletter form, smooth scroll, mask reveals
